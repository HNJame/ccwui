说明文档

一款基于 jQuery UI 开发的时间日期组件

使用方法与参数说明：
$( "selector" ).datetime({
showOn:"focus", // 触发时间控件的方式，可选参数 "focus"、"button" 或者 "both"
buttonImage: "images/icon/icon-calendar.png", // icon按钮的图片路径
buttonImageOnly: false, //
dateFormat: "yyyy/MM/dd", //文本框中显示日期的格式
time: true, // 是否可以选择具体时间
seconds: false, // 是否可以选择秒数，只有当time属性为true时有效
changeYear: true, // 是否可以通过下拉框的方式选择年份
changeMonth: false //是否可以通过下拉框的方式选择月份
});

更多细节请查看 Demo。
